Algebraic Identities

Three identities that come straight from the distributive property — and that turn hard squarings and products into quick mental maths.

Key Concept The three identities

(a + b)² = a² + 2ab + b²
(a − b)² = a² − 2ab + b²
(a + b)(a − b) = a² − b²

Each comes from multiplying out the brackets. The only difference between the first two is the sign of the middle term.

Using them for fast calculation

•
Split a number into an easy sum or difference. 55² = (50 + 5)² = 2500 + 500 + 25 = 3025.
•
99² = (100 − 1)² = 10000 − 200 + 1 = 9801.
•
397 × 403 = (400 − 3)(400 + 3) = 160000 − 9 = 159991.
Bonus identities: 2(a² + b²) = (a + b)² + (a − b)², and a² = (a + b)(a − b) + b² — the second is a neat trick for squaring (e.g. 31² = 32 × 30 + 1 = 961).
Prerequisite refresher: "ab" means a × b, and the "middle term" 2ab is twice the product of the two parts.

Common Traps Where marks slip away

Trap 1 — Don't forget the middle term. (a + b)² = a² + 2ab + b², NOT a² + b². The 2ab is the most-missed term.
Trap 2 — Sign of the middle term. In (a − b)² it is −2ab; in (a + b)² it is +2ab.
Trap 3 — Square each whole part. In (6x + 5)², (6x)² = 36x², not 6x².
Trap 4 — Difference of squares needs a − b. (a + b)(a − b) = a² − b² only when the two brackets are a sum and the matching difference.
Slip 5 — (a − b)² = (b − a)². Squaring removes the sign, so both give the same value.

Worked Examples Model answers — write them like this

Example A. Expand (6x + 5)².
Step 1:Use (a + b)² with a = 6x, b = 5.
Step 2:(6x)² + 2(6x)(5) + 5² = 36x² + 60x + 25.
∴ 36x² + 60x + 25
Example B. Expand (3j + 2k)².
Step 1:a = 3j, b = 2k.
Step 2:(3j)² + 2(3j)(2k) + (2k)² = 9j² + 12jk + 4k².
∴ 9j² + 12jk + 4k²
Example C. Find 99² using an identity.
Step 1:99 = 100 − 1, so use (a − b)² with a = 100, b = 1.
Step 2:100² − 2(100)(1) + 1² = 10000 − 200 + 1.
∴ 99² = 9801
Example D. Find 397 × 403 using an identity.
Step 1:397 = 400 − 3 and 403 = 400 + 3, so use (a − b)(a + b) = a² − b².
Step 2:400² − 3² = 160000 − 9.
∴ 397 × 403 = 159991
