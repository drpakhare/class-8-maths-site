Zero & Negative Exponents

What does a power of 0 mean? Or a negative power? These rules let the laws of exponents work for every integer — and they appear in lots of 1-mark questions.

Key Concept Three rules to remember

1.
Zero exponent: any non-zero number to the power 0 equals 1.   n⁰ = 1 (n ≠ 0).   e.g. 7⁰ = 1, (−5)⁰ = 1.
2.
Negative exponent = reciprocal:   n⁻ᵃ = 1 ⁄ nᵃ.   e.g. 3⁻² = 1⁄3² = 1⁄9.   Also 1 ⁄ n⁻ᵃ = nᵃ.
3.
Power of a fraction:   (a⁄b)ⁿ = aⁿ ⁄ bⁿ,   and a negative power flips the fraction: (a⁄b)⁻ⁿ = (b⁄a)ⁿ.
Big idea: a negative exponent does NOT make a number negative. It means "take the reciprocal." 3⁻² = 1⁄9, which is positive.
All five laws still apply with zero and negative exponents. For example 3⁵ ÷ 3⁻⁶ = 35−(−6) = 3¹¹.
Why n⁰ = 1? Because nᵃ ÷ nᵃ = na−a = n⁰, and any number divided by itself is 1.

Common Traps Where marks slip away

Trap 1 — Negative power ≠ negative number. 2⁻³ = 1⁄8, not −8 and not −6.
Trap 2 — n⁰ = 1, not 0. Any non-zero base to the power 0 is 1.
Trap 3 — Forgetting to flip the fraction. (2⁄5)⁻² = (5⁄2)² = 25⁄4. Flip first, then square.
Trap 4 — Subtracting a negative exponent. 3⁵ ÷ 3⁻⁶ = 35−(−6) = 3¹¹, not 3⁻¹.
Slip 5 — Sign with fractions. (−3⁄4)⁻³ = (−4⁄3)³ = −64⁄27 (odd power keeps the minus sign).

Worked Examples Model answers — write them like this

Example A. Find the value of 3⁻².
Step 1:Negative exponent → reciprocal: 3⁻² = 1 ⁄ 3².
Step 2:3² = 9, so 3⁻² = 1 ⁄ 9.
∴ 3⁻² = 1⁄9
Example B. Find the value of 1 ⁄ 4⁻².
Step 1:1 ⁄ n⁻ᵃ = nᵃ, so 1 ⁄ 4⁻² = 4².
Step 2:4² = 16.
∴ 1 ⁄ 4⁻² = 16
Example C. Find the value of (2⁄5)⁻².
Step 1:Negative power flips the fraction: (2⁄5)⁻² = (5⁄2)².
Step 2:(5⁄2)² = 25 ⁄ 4.
∴ (2⁄5)⁻² = 25⁄4
Example D. Simplify 3⁵ ÷ 3⁻⁶.
Step 1:Same base, dividing → subtract exponents: 5 − (−6).
Step 2:5 − (−6) = 5 + 6 = 11.
∴ 3⁵ ÷ 3⁻⁶ = 3¹¹
Example E. Express 16⁻² as a power with base 2.
Step 1:Write 16 as a power of 2: 16 = 2⁴.
Step 2:16⁻² = (2⁴)⁻² = 2⁴ˣ(⁻²) = 2⁻⁸.
∴ 16⁻² = 2⁻⁸
Example F. Simplify 2⁻⁴ × 2⁷ and write with a positive exponent.
Step 1:Same base, multiply → add exponents: −4 + 7 = 3.
Step 2:So the answer is 2³ = 8.
∴ 2⁻⁴ × 2⁷ = 2³ = 8
