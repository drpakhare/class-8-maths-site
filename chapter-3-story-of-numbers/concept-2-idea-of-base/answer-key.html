Section C Long answer

Q10
(i) 400 in base 5   (ii) 1210 in base 4. 100 = 4×25 (base 5); 100 = 1×64 + 2×16 + 1×4 + 0 (base 4).
Q11
16 = 100 in base 4. 16 is exactly the landmark number 4² = 16, so it is 1×16 + 0×4 + 0×1.
Q12
Because landmark numbers are powers, their product is another landmark number (e.g. 10² × 10³ = 10⁵). The Roman system has irregular landmark sizes, so multiplication there is much harder.
