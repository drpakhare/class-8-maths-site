Section C Long answer

Q10
Cube root = 15. 3375 = (3×3×3)(5×5×5), so ∛3375 = 3 × 5 = 15.
Q11
Multiply by 60. 3600 = 2⁴ × 3² × 5²; need 2² × 3 × 5 = 60 to complete the triples (3600 × 60 = 216000 = 60³).
Q12
Divide by 4; ∛64 = 4. 256 = 2⁸; remove 2² → 2⁶ = 64 = 4³.
